## Milestone 1

**Name**
BCYX-SWAP Testnet Prototype

**Target Date**
4 weeks after grant start

**Description**
Deliver a working BTC ↔ sBTC coordination prototype implementing BCYX's core primitives: commitment-based swap intents, shielded coordination pools, intent matching, nullifier protections, and atomic coordination logic. The implementation will operate in local/test environments using mocked settlement adaptors where necessary.

**Success Criteria**

* Public GitHub repository updated with the prototype implementation.
* Users can create and submit private swap commitments.
* Coordination nodes successfully match compatible intents.
* Atomic coordination flows execute successfully in test environments.
* Technical documentation describing the architecture and swap lifecycle is published.
* Public demo video showing the prototype in operation.

## Milestone 2

**Name**
End-to-End BCYX-SWAP Demonstration

**Target Date**
8 weeks after grant start

**Description**
Validate the complete BCYX-SWAP Proof-of-Concept by demonstrating private BTC ↔ sBTC coordination from intent creation through successful coordination completion. Publish implementation findings, benchmarks, and lessons learned to assess the viability of ephemeral coordination as a primitive for Bitcoin interoperability.

**Success Criteria**

* End-to-end swap coordination demo completed and publicly available.
* Batch coordination and accumulator mechanisms demonstrated.
* Benchmarks documenting coordination performance and limitations published.
* Architecture documentation and specifications finalized.
* Public technical report summarizing outcomes, trade-offs, and future directions released.

**Adoption Metric**
At least **3 external developers, reviewers, or ecosystem contributors** provide feedback through GitHub issues, discussions, pull requests, or public technical review of the BCYX PoC.

---

### Total Request

* **Duration:** **~8 weeks**
* **Primary Outcome:** Validation of **ephemeral private coordination for BTC ↔ sBTC interoperability** as an open-source primitive for the Stacks ecosystem.
